# MarketPulse-Kotak
MarketPulse Kotak is a desktop trading terminal built to interact with the Kotak Neo API. It provides quick access to orders, positions, holdings, and margin details with a clean interface designed for speed and usability.

Advantage : 
1) Skip brokerage on orders Completely (Note :Only Order costs )
2) Live Ticks and Realtime Trading Dashbaord ,Clean UI
